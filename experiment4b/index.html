Description

An 'infinite world' is meant to create a nearly limitless experience to give a user a
technically new experience everytime they dive in. For this experience, I replicated
Minecraft's snowy mountain biome because most of the time when I think about
wandering, it's almost always in a snowy plane, climbing slopes to explore a cold, yet
curious land. In respect to my source material, you can leave your mark physically by
digging out the ground to access underground water, making your mark on the land.

Technical

The moutainous terrain was created with a perlin noise map which was created by
being fed a world seed. It is then fed through a finite distance algorithm which
determines how high and how low each part of the map should look. I then created
fake 3d using a 3/4th view in order to fake a 'z axis' (up/down in 3d space), using the
combination of the x and y planes, side to side on the screen and up and down in 2d
space. This fake up and down reflects the changes in physical altitude alongside a
gradient that colors the the terrain based on the illusioned altitude. Finally, using the
hash value of each world coordinate, I used an algorithm to populate the highest parts
of the terrain with trees and cabins, both of which are sprites created by yours truly.
